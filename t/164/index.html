Nom du projet :
VIRMED

Description du besoin (cas clinique auquel répond votre projet) :
Je suis face à une situation dans laquelle je dois remplacer le.s médicament.s d’une prescription avec un similaire, tout en tenant compte du contexte clinique du patient.

Membres de l’équipe (elle peut s’agrandir) :

  • @Yvon (MD, Medical Computer Scientist)
  • @scossin (MD, Medical Computer Scientist, NLP Expert)
  • @Roua (Pharmacist)
  • @Myriam
  • @thierry.geufroi
  • @Wiil

Mentor :
Mirojane Mohammad (HAS)

Description du projet (20 lignes maximum) :

Notre solution est d’afficher les médicaments similaires classés selon leur degré de similarité avec le médicament recherché, mais aussi d’afficher leur différence par rapport à ce médicament :

  • la forme galénique est différente (comprimé / gélule ; effervescent ou non)
  • le dosage est différent (1000mg / 500mg)
  • la voie d’administration différente (orale / rectale)
  • la substance est différente mais la composition en substance active est identique (Abacavir / Sulfate d’abacavir)
  • la substance est différente mais l’indication est identique

L’utilisateur pourra choisir au besoin les critères de dissimilarité souhaités.
