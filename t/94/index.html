Pour les personnes intéressées par les résumés des caractéristiques du produit (RCP) des médicaments, l’équipe de recherche en informatique Erias avait développé des scripts pour structurer les différentes sections (indications, posologie, effets indésirables…) d’un RCP.
Ces scripts sont open-source : GitHub - scossin/FrenchSPC: French Summaries of Product Characteristics (SPC) / Résumés des Caractéristiques du Produit (RCP) et
les fichiers Json peuvent être téléchargés à ce lien :
http://91.121.106.228:8087/SPC/
